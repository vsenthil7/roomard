# Roomard logo set

Brand-consistent with the pitch deck + cover images (deep teal `#0a4a3f`,
signal accent `#10b981`, Inter wordmark). Generated by `scripts/build_logo.py`
(re-run to regenerate).

The mark is a rounded **arch / room threshold** enclosing a **memory dot** —
the remembered guest sitting within the room. It nods to what Roomard is: a
guest-memory engine.

| File | What | Use |
|---|---|---|
| `roomard-logo-mark.svg` | Vector glyph only, transparent | Scalable mark, app source-of-truth |
| `roomard-logo-lockup.svg` | Glyph + "Roomard" wordmark, horizontal | Headers, README, deck |
| `roomard-icon-tile.svg` | Glyph on a teal rounded tile (mint glyph) | App / favicon source |
| `roomard-logo-mark-512.png` | 512×512 mark, transparent | General raster use |
| `roomard-logo-mark-1024.png` | 1024×1024 mark, transparent | Hi-res raster |
| `roomard-logo-lockup-1600.png` | 1600-wide lockup, transparent | Submission header, social |
| `roomard-icon-32.png` | 32×32 tile | Favicon |
| `roomard-icon-180.png` | 180×180 tile | Apple touch icon |

Regenerate: `python scripts/build_logo.py`
(prefers `cairosvg` for crisp SVG→PNG; falls back to a Pillow-drawn equivalent
if cairosvg isn't installed, so it always produces the PNGs.)
